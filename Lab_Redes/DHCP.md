# DHCP

DHCP es un protocolo que permite a los hots de una red obtener su dirección IP de forma automática. Un servidor DHCP debe mantener un pool de direcciones (Conjunto de direcciones IP que pueden ser asignadas a un host mediante este protocolo) Cuando un host pide una direcci´on mediante DHCP, la dirección se retira de forma temporal del pool hasta que venza el tiempo de préstamo sin que se produzca una renovación (lease time) o hasta que el host libere la dirección.

# DNS
El sistema de nombres de dominio (DNS) es un protocolo fundamental que permite traducir nombres de dominio legible en direcciones IP que las computadoras pueden entender.
